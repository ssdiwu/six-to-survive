# v0.0.2 版本实施方案 — 手感打磨 + 玩法深化

> 版本：v0.0.2
> 创建日期：2026-07-16
> 基于版本：v0.0.1（首个可玩版本）
> 目标：提升核心循环手感、视觉听觉反馈、玩法深度，为录屏 demo 做打磨

## 目标

v0.0.1 已跑通完整一局循环，v0.0.2 在此基础上做两类工作：
- **A 组 观感打磨**：让游戏看起来、听起来、打起来更爽
- **B 组 玩法深化**：补全敌人/资源/词条系统，让 build 多样性和决策深度到位

## In Scope

### A 组：观感与手感

| # | 内容 | 说明 |
|---|---|---|
| A1 | 空格弹 / 回车弹 | 空格打出大一号"␣"字符弹，伤害+1；回车打出"⏎"发送弹带拖尾穿透1个敌人 |
| A2 | 精力攻速连续梯度 | 替代 v0.0.1 的二态（能打/不能打），改为：>80 攻速 100%，30-80 线性衰减，<30 攻速 50%，0 不能打 |
| A3 | SAN<30 视觉效果 | 屏幕轻微色差偏移、20% 字符弹变乱码字符、背景噪点 |
| A4 | Web Audio 打字音 | 按字母键播放合成咔哒声（Web Audio API 程序化生成，无外部资源） |
| A5 | JetBrains Mono 字体 | 通过 Google Fonts CDN 加载，字符弹/HUD 数字用等宽字体，系统字体兜底 |
| A6 | 粒子拖尾特效 | 子弹和敌人死亡时的拖尾/爆散粒子（轻量，不影响性能） |

### B 组：玩法深化

| # | 内容 | 说明 |
|---|---|---|
| B1 | 新增 3 种敌人 | 飞书文档@评论（贴边停留放小弹）、飞书审批待办（贴边扣精力）、飞书 OKR/季末对齐（精英慢厚血） |
| B2 | KPI 事件 | KPI<20 触发领导约谈（屏幕顶部头像压下，期间伤害翻倍、禁道具）；KPI>80 优秀员工攻速 +20% buff |
| B3 | 回旋锅机制 | 甩锅贴反弹的弹幕（IM/会议）飞出屏幕边缘后沿原路径回飞一次敌对弹，玩家要敲掉 |
| B4 | 词条池扩到 30 个 | 补全 PRD 里剩余词条（耳机降噪、会议摸鱼、需求记录员、甩锅闭环、全公司静默升级、站会、需求考古学、老板附体、股权承诺书、N+1 大礼包等），保持四级稀有度 |
| B5 | 稀有度动态权重 | 随等级升高，Common 概率递减、Epic/Legendary 概率递增 |

### Bug 修复（前置）

- 敌人速度跨帧率不一致（统一用 px/秒）
- 升级选卡可能出现重复词条（去重）
- 离职缓冲道具只检查 HP=0 不检查 SAN=0
- resize 后敌人位置视觉瞬移（敌人坐标相对屏幕中心归一化）

## Out of Scope（推迟）

- 三阶段 Boss（横幅压顶/质疑弹/deadline buff）→ v0.3
- 道具栏满替换机制 → v0.1.1（小补丁，可后做）
- 升级一次多拿牌 → v0.2
- 斜 45° 透视视觉（已决策不用，见 ADR 0002）

# v0.0.3 版本实施方案 — 三资源手感重构

> 版本：v0.0.3
> 创建日期：2026-07-16
> 基于版本：v0.0.2

## 目标

重构资源系统为"压力/体力/毕业进度"三资源模型，让每一条都有不可替代的博弈意义；修复 v0.0.2 中精力无感、HP/SAN 重复、KPI 条不直观等问题；提升敌人血量到更有挑战性的水平，扩充词库到全敌人 30+ 条职场黑话。

## 核心变更

### 1. 三资源重构

| 旧 | 新 | 方向 |
|---|---|---|
| HP + SAN（两条资源） | **压力**（单条，0-100） | 高=危险，被打涨/不打降/摸鱼降/满=崩溃 |
| 精力 | **体力**（0-100） | 低=危险，敲键降/自动回/空=攻速减半 |
| KPI | **毕业进度**（0-100） | 高=危险，不打涨/敲键降/满=优化谈话 |

博弈三角：
- 狂敲 → 体力空 → 攻速减半 → 清不动 → 被迫停
- 停手 → 压力回、体力回 → 保命
- 但停手 → 毕业进度涨 → 被优化谈话 → 伤害翻倍禁道具

### 2. SAN 低 fumble → 压力高 fumble

- 压力 ≥ 70% 时：25% 打出随机错字母 + 方向偏移 ±1.5 rad，15% 打出乱码
- 压力 ≥ 50% 时：15% 概率错字，轻微方向偏移
- 情绪稳定器词条：压力 ≥70% 时仍可打字但攻速 ×0.7

### 3. 敌人 HP 提升

| 敌人 | v0.0.2 | v0.0.3 |
|---|---:|---:|
| 飞书 IM | 2 | 3 |
| 同事@ | 3 | 4 |
| 会议邀请 | 8 | 12 |
| 文档@评论 | 5 | 7 |
| 审批待办 | 3 | 4 |
| OKR 对齐 | 15 | 20 |

### 4. 词库扩充

- IM：保留 101 条
- 会议：32 条（含 kickoff/对齐会/复盘会等）
- 同事：30 条（含线上 bug/环境挂了/谁部署的等）
- 评论：32 条（含体验不好/再改改/再出个方案等）
- 审批：30 条（含离职交接/晋升答辩/加班审批等）
- OKR：30 条（含战略解码/人才盘点/绩效校准等）

### 5. 其他修复

- HiDPI 字体发虚：按 devicePixelRatio 缩放 canvas
- HUD 四条改三条，布局重新调优不重叠
- 领导约谈条（现"优化谈话"）位置下移避开时间条
- 升级指数曲线 `15×1.15^level` 保留
- 离职缓冲回 20 压力（不再回 HP/SAN 旧字段）
- 摸鱼奶茶使用期间压力快速下降（-8/s），毕业进度涨 2 倍

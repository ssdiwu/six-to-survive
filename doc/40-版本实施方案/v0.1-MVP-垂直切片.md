# v0.1 MVP 垂直切片实施方案

> 版本：v0.1
> 创建日期：2026-07-16
> 对应 PRD：`doc/00-产品与原则/PRD.md`
> 对应 ADR：`doc/决策档案/0001`、`0002`、`0003`

## 目标

交付一个可完整游玩的单文件 HTML 游戏，从标题页开始到 18:00 胜利或 HP/SAN 归零失败，跑通"打字清弹→升级选卡→道具使用→下班准备期→结算"的完整循环。本次不追求内容完整，只验证核心手感和数值节奏。

## 范围

### 本次实现（In Scope）

- 5 个游戏状态：TITLE / PLAYING / PAUSED / LEVEL_UP / GAME_OVER
- 纯俯视角 Canvas 2D 渲染，正方形工位中央
- 字母键 A-Z 打字攻击，攻速限频，精力耗尽不能打字
- 3 种敌人：飞书 IM 消息、飞书会议邀请、同事@你问问题
- 4 条资源条：HP / SAN / 精力 / KPI（HUD 显示 + 数值逻辑）
- 时间系统：9:00→18:00，暂停/升级时停，摸鱼奶茶期间敌人冻结但时间继续
- 6 个道具：摸鱼奶茶、甩锅贴、咖啡罐、PPT 装甲、降噪耳机、离职缓冲
- 12 个词条：6 Common / 3 Rare / 2 Epic / 1 Legendary
- 5 选 1 升级：每 15 杀触发，卡池按权重随机
- 老板查岗：14:30 固定触发一次
- 下班准备期：17:30 起敌人密度 ×3
- 胜利/失败结算页（鼠标操作）
- 角色在工位内的状态演出动画（坐姿/踱步/喝茶/受击）

## 键位表

| 键 | 行为 |
|---|---|
| A–Z | 打出字符弹，朝最近敌人飞行 |
| 1–5 | 使用对应栏位道具 |
| Esc | 暂停 / 继续 |
| Space / Enter | 预留（静默，v0.0.2 实现） |
| 0 / 6–9 / 标点 | 静默 |
| Tab / Backspace | 静默 + `preventDefault()` |
| F1–F12 / Ctrl / ⌘ / Alt | 透传给浏览器 |
| 鼠标 | 标题页、暂停页、结算页按钮交互；升级选卡可点 |

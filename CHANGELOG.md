# 变更日志

本文件记录《Six to Survive · 撑到六点》面向玩家的显著变化。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本以 Git 标签为准。

## [Unreleased]

## [0.0.5] - 2026-07-18

### Added
- **时间点波次事件**：10:00 早会（会议邀请爆发）、14:00 午休结束消息轰炸（15条 IM）、17:00 下班前高压（全类型混合+BOSS级）
- **经验条 UI**：时间条下方显示当前等级和到下一级进度
- **清屏词条**：「深呼吸爆」（Epic，深呼吸结束全屏 20 伤害）、「锅甩远点」（Legendary，甩锅伤害×3）

### Changed
- HUD 排布顺序：时间条 → 经验条 → 三资源条
- 波次触发有屏幕闪色提示

## [0.0.4] - 2026-07-18

### Added
- **主动/被动系统重构**：去掉随机道具系统，改为 5 个固定主动技能（咖啡/Ctrl+V/摸鱼/深呼吸/甩锅），靠不同指标充能（咖啡与 Ctrl+V 按键数、摸鱼体力消耗、深呼吸压力上涨、甩锅毕业进度上涨）；35 个被动词条分 4 级稀有度（Common 5 级 / Rare 3 级 / Epic 2 级 / Legendary 1 级，重复拿升级）
- **8-bit 背景音乐**：激昂方波+贝斯+鼓点，128 步 8 段 ~23s 循环；暂停/百科/结算时降音量 25% 不停
- **百科全书**：标题页和暂停菜单都能进入，展示所有技能/词条/机制说明
- **全屏 AOE 甩锅**：按 5 全屏敌人 10 伤害，屏幕闪橙，毕业进度清零
- **受击反馈**：0.5s 无敌帧 + 屏幕震动
- **桌面打包**：Tauri 原生 .app/.dmg（macOS），Go 静态服务器版 .exe（Windows）
- 键程优化改体力消耗减；新增「功能饮料」词条（体力回复+25%/级）

### Changed
- 游戏时长从 10 分钟缩短为 5 分钟（时间流速 ×2）
- 三资源顺序：体力→压力→毕业进度
- 压力条从"挨打涨"改为持续打字 +1/s、不打 -0.8/s（输出期间涨、不打缓降，博弈更紧张）
- 咖啡效果：攻速翻倍 → 双发追踪弹 + 不耗体力
- Ctrl+V：3 秒内每键打最近 3 个敌人（多发追踪，非扇形）
- 深呼吸：主动技能，5 秒内 -4/s 压力
- 双选金词条：拿到后本次升级 5 选 2
- 去掉攻速硬上限，手速=攻速
- 角色表情按压力高低切换
- 词条升级指数曲线保留（15×1.15^N）
- 所有道具效果改为词条被动效果（咖啡成瘾、耳机降噪、PPT装甲、甩锅大师等）
- 耳机降噪改为敌方弹幕减速 7%/级

### Fixed
- 死亡判定 maxP 变量作用域错误导致被碰后不弹出结算
- spawnParticles 函数缺失导致 tick 中断、压力/毕业进度不变化
- TIME 常量丢失导致时间条显示 NaN
- HiDPI 字体发虚（devicePixelRatio 缩放）
- 标题页错误渲染 HUD/技能栏/工位
- 熬夜修仙仅在体力 0 打出子弹时 +1 压力
- 体力 clamp 不会负数
- 回旋锅机制删除
- 攻速词条改为「攻速上限」（手速快玩家不会被限频）
- 离职缓冲改为金词条「情绪稳定器」+「被动反击」

## [0.0.3] - 2026-07-16

### Changed
- **三资源系统重构**：HP+SAN 合并为「压力」（高危险），精力改为「体力」，KPI 改为「毕业进度」（高危险）
- 压力：被打/硬直/甩锅涨；不敲/摸鱼/词条降；满=精神崩溃
- 体力：敲键消耗、自动回复；空=攻速减半
- 毕业进度：不敲涨、敲键降；满=优化谈话（5秒伤害翻倍、禁道具）
- 博弈闭环：狂敲→体力空→攻速降→被迫停→毕业进度涨→约谈
- 精力消耗从 0.5 提到 1.2/键，回复 3/s→1/s，让体力真正有意义
- 敌人 HP 提升：IM 2→3、同事 3→4、会议 8→12、评论 5→7、审批 3→4、OKR 15→20
- 敌人生成密度加密
- HiDPI (devicePixelRatio) 缩放修复字体发虚
- HUD 从四条改三条，间距调整避免重叠

### Added
- 压力高 fumble：≥50% 有几率错字偏移，≥70% 高几率错字/乱码/方向偏
- 全敌人扩充词库到 30+ 条职场/飞书黑话（IM 101、会议32、同事30、评论32、审批30、OKR30）
- 摸鱼奶茶期间压力快速下降 -8/s
- 优秀员工 buff：毕业进度 <20 时攻速 +20%

### Fixed
- 升级选卡重复问题（usedIds Set）
- 离职缓冲覆盖 SAN=0 崩溃
- 结算时长冻结
- 领导约谈条位置避开时间条

## [0.0.2] - 2026-07-16

### Added
- 空格/回车特殊弹（空格伤害+1、回车穿透1个敌人）
- 精力攻速连续梯度（>80%满速、30-80%线性衰减、<30%半速）
- SAN<30 视觉效果（色差覆盖、乱码弹、背景噪点、暗角）
- Web Audio 程序化打字音
- JetBrains Mono CDN 等宽字体
- 子弹拖尾粒子、敌人死亡爆散火花粒子
- 结算页保存 1200×630 战绩分享图 PNG
- 三种新敌人：文档@评论（贴边放小弹）、审批待办（贴边扣精力）、OKR 对齐（精英高血量）
- KPI 事件：KPI<20 触发领导约谈（伤害翻倍、道具禁用），KPI>80 攻速+20%
- 回旋锅机制：甩锅贴反弹的弹幕飞出屏幕后沿原路径回飞
- 25 个词条（10 Common / 8 Rare / 5 Epic / 3 Legendary），含 N+1 大礼包、股权承诺书、双选、老板附体等
- 稀有度动态权重：随等级降低 common 概率、提升 Epic/Legendary 概率

### Fixed
- 升级选卡不再出现重复词条/道具
- 离职缓冲道具在 SAN=0 时也生效

## [0.0.1] - 2026-07-16

### Added
- 首个可玩版本：浏览器 2D 弹幕 Roguelike，单文件 HTML 交付
- 完整一局循环：标题页 → 9:00–18:00 生存 → 胜利/失败结算
- A–Z 键打字攻击，字符弹飞向最近飞书消息
- 三种敌人：飞书 IM 消息（飞书文字气泡）、会议邀请（硬直）、同事@你问问题（可被甩锅反弹）
- 四条资源：HP / SAN / 精力 / KPI，带顶部 HUD 显示
- 主动道具系统：1–5 数字键栏位，6 个道具（摸鱼奶茶、甩锅贴、咖啡罐、PPT 装甲、降噪耳机、离职缓冲），可堆叠使用次数
- 12 个被动词条 + 10% 概率道具卡的 5 选 1 升级，每 15 击杀触发
- 14:30 老板查岗事件（探照灯横扫、禁用道具、伤害翻倍）
- 17:30 下班准备期（敌人密度 ×3）
- Esc 暂停、结算页统计（时长/击杀/道具/词条/连击）
- 工位正方形内角色状态演出（打字/摸鱼/受击表情）

### Fixed
- 结算页本局时长不再持续增长，进入结算时冻结
- 敌人渲染从圆点改为飞书风圆角气泡 + 随机中文文案

### Changed
- 项目英文名由 Nine to Survive 改为 Six to Survive，与"撑到六点"主题一致
